Syntax for sublime


ctrl-F             Select all occurences
ctrl-H             Replacement value
ctrl-U             Undo selection

ctrl-enter         Insert line below the current line
ctrl-shift-enter   Insert line above the current line

ctrl-shift-up      Move current line up one line
ctrl-shift-down    Move current line down one line
